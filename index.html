Josh by yifanlu

Extended Windows console library for Java

This is a Windows ONLY console library for Java. The goal is to port ALL native console functions found in wincon.h in the Windows API to Java, with the top priority being the most common functions.

Dependencies

Java Windows

Install

1. Copy the JoshCore.dll to wherever your .class is 2. Copy either the .jar (from /JAR) or the "com" folder (from /CLASS) to where your .class is
